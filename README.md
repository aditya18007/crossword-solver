# crossword-solver
Solve a 10x10 Crossword using backtracking. 
My solution to hackerrank problem. https://www.hackerrank.com/challenges/crossword-puzzle/problem

### Sample input
```
+-++++++++
+-++++++++
+-++++++++
+-----++++
+-+++-++++
+-+++-++++
+++++-++++
++------++
+++++-++++
+++++-++++
LONDON;DELHI;ICELAND;ANKARA
```

### Output
```
+L++++++++
+O++++++++
+N++++++++
+DELHI++++
+O+++C++++
+N+++E++++
+++++L++++
++ANKARA++
+++++N++++
+++++D++++
```
